# GetVela Variation Generator

A local web application/script designed to automatically generate product variations and export them directly into GetVela-formatted data sheets. This tool helps streamline multi-tier product options (such as size and color matrix variations) while respecting character volume limits.

## 🚀 Features
- **Multi-Tier Variation Generation:** Easily combine different attributes like colors, sizes, and styles.
- **GetVela Compatible:** Outputs structured data sheets ready to import directly into GetVela.
- **Character Limit Validation:** Automatically checks and restricts variation names according to platform rules.
